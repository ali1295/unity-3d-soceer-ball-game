# ⚽ Unity 3D Soccer Ball Game

A simple and interactive **3D Soccer Ball Game** built using **Unity Engine** and **C#**.  
This project demonstrates basic football mechanics such as ball physics, player interaction, and goal scoring using Unity’s physics system.

---

## 🎮 Gameplay Overview

In this game, the player interacts with a soccer ball in a 3D environment and aims to control, push, or kick the ball towards a goal.

The project focuses on **realistic ball physics and collision handling**, which are essential parts of football-style games. :contentReference[oaicite:0]{index=0}

---

## 📌 Features

- ⚽ Realistic soccer ball physics (Rigidbody)  
- 🎮 Player movement and interaction  
- 🥅 Goal detection system  
- 🔁 Continuous gameplay loop  
- 💥 Collision-based interactions  
- 🎨 Basic 3D environment setup  

---

## 🛠️ Built With

- Unity Game Engine  
- C# Scripting  
- Unity Physics System (Rigidbody & Colliders)  
- Visual Studio / Rider
